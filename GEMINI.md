Project rule:
This project is Battery Engineering Hub, a public SEO microtool website. Never include confidential company data, customer data, internal battery datasets, proprietary SOH algorithms, AYK-specific workflows, or private formulas.

Build rules:
- Use Astro for static SEO pages.
- Use Tailwind CSS v4.
- Follow DESIGN.md as the single source of visual truth.
- Every calculator must expose public formulas clearly.
- Every calculator must include disclaimer text.
- Every page must be mobile-first, fast, and SEO-ready.
- Use reusable calculator components.
- Keep logic in transparent TypeScript utility files.
- No backend in v1.
- No database in v1.
- No login in v1.

- SEO title

- Result card
- Formula section
- Worked example
